# Testing

When building a web application it's important to be able to test the behaviour of your system to ensure it does what you expect.

A Keystone system presents a GraphQL API to the world, which web-browers and other clients can communicate with.

In this guide we'll show you how to use the `@keystone-next/testing` package to write tests to check the behaviour of your GraphQL API.
